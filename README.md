# ezreadme
Easiest way to generate a simple README for your project
#### [Homepage](https://github.com/thidasapankaja/ezreadme)
#### Prerequisites
 ``` 
npm
```
#### Installation
Run particular commands according to your package manager
```
// using npm
npm install ezreadme

// using yarn
yarn add ezreadme
```
#### Usage
```
generate-readme
```
